Este arquivo tem como objetivo principal mostrar o uso do ggplot2 para criação de gráficos e boas práticas.

Criação de gráficos: GGPLOT2

Vamos carregar a biblioteca ggplot2 e outros pacotes

Lendo o conjunto de dados

Básico

O ggplot2 funciona com construção de “camadas” (layers). O comando {r addition} ggplot() inicia um gráfico vazio.

ggplot()

A partir deste gráfico vazio, podemos adicionar camadas de pontos, linhas, barras, caixas, entre outros. Dê uma olhada no manual de referência. Outros como tidyverse, ou Curso de R podem ajudar bastante! Explore o máximo.

Adicionando layers

De maneira geral, a sintaxe que vamos usar é {r addition} geom_yyy(data = dados, aes(x = x, y = y)), sendo “line” para linhas, “point” para pontos e assim por diante.

Por exemplo, para criar um plot de peso x altura temos

ggplot() +
  geom_point(data = dados, aes(x = height, y = weight))

Se quisermos que os pontos tenham uma cor em particular, usamos

ggplot() +
  geom_point(data = dados, aes(x = height, y = weight), color = "blue")

# Usar o dataset iris, disponível por padrão no R
data(iris)

# Criar um split treino/teste
set.seed(123)
split <- initial_split(iris, prop = 0.8)
train <- training(split)
test <- testing(split)

# Definir o modelo (regressão logística multinomial)
modelo <- multinom_reg() %>%
  set_engine("nnet") %>%
  set_mode("classification")

# Criar o workflow
workflow <- workflow() %>%
  add_model(modelo) %>%
  add_formula(Species ~ .)

# Ajustar o modelo
ajustado <- fit(workflow, data = train)

# Previsão e avaliação
pred <- predict(ajustado, new_data = test) %>%
  bind_cols(test)

# Métricas de desempenho
metrics(pred, truth = Species, estimate = .pred_class)
